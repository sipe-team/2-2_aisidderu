# LUMA

최근에 사용해본 비디오 생성 모델

- 장점
  - 프롬프트로 쉽게 생성 가능
  - 이미지를 입력할 수 있음
  - 하루에 5번 무료
- 단점
  - 너무 느림
  - 생성 퀄리티가 프롬프트에 크게 좌지우지 하는 듯함.
  - 이미지를 입력으로 넣더라도 자연스러운 동영상이 나오지는 않음
